This folder lists Drivers/Plugins for Experica to function properly, install them as needed.


InpOutBinaries_1501: Windows kernel mode driver for parallel port.

SpikeGLXDotnet40Installer_web: Plugin for coordinate with SpikeGLX data acqusition system.

XippmexDotNet40Installer_web: Plugin for control Ripple data acqusition system.
